Privacy Policy for AI-Job-Application

AI-Job-Application helps users prepare job applications by analyzing job postings, generating cover letter drafts, and creating user-reviewed autofill suggestions.

Data Collected

The extension may process the following data when the user chooses to use its features:

1. Personal information entered by the user, such as name, email address, phone number, location, LinkedIn, GitHub, website, skills, resume summary, work experience, and cover letter template.
2. Website content from job posting pages and application forms, such as job titles, company names, job descriptions, visible page text, and form field labels.

How Data Is Used

The data is used only to provide the extension's job application assistance features, including:

- Saving the user's profile locally
- Extracting resume details from an uploaded PDF
- Analyzing job fit
- Generating cover letter drafts
- Creating autofill suggestions for application forms

Data Storage

User profile data and settings are stored locally in Chrome storage. The extension does not sell user data.

AI Provider Requests

AI-powered features require the user to configure their own AI provider and API key. When the user explicitly triggers an AI feature, relevant profile data, resume text, job posting content, or form field information may be sent to the selected AI provider to generate the requested result.

Data Sharing

The extension does not sell user data and does not transfer user data for purposes unrelated to its single purpose. Data is shared with the user's selected AI provider only when required to perform a user-triggered AI feature.

Remote Code

The extension does not execute remote JavaScript or WebAssembly.
